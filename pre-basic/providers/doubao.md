### 

[hashtag](#huo-qu-api-key)

获取API Key

*   点击侧栏下方的 [API Key管理arrow-up-right](https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey)
    
*   创建 API Key
    

![](https://docs.cherry-ai.com/~gitbook/image?url=https%3A%2F%2F3562065924-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F0Ut5BptC3t8CtSU1UWpM%252Fuploads%252Fgit-blob-0f0e86cfcba964a00d17b9441c71bd4bbd821881%252Fimage%2520%286%29%2520%282%29.png%3Falt%3Dmedia&width=768&dpr=4&quality=100&sign=6c010061&sv=2)

*   创建成功后，点击创建好的 API Key 后的小眼睛打开并复制
    

![](https://docs.cherry-ai.com/~gitbook/image?url=https%3A%2F%2F3562065924-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F0Ut5BptC3t8CtSU1UWpM%252Fuploads%252Fgit-blob-0b48dd8da1dad5f57bcc27ebc724c2db55440962%252Fimage%2520%287%29%2520%282%29.png%3Falt%3Dmedia&width=768&dpr=4&quality=100&sign=80b44e99&sv=2)

*   将复制的 API Key 填入到 CherryStudio 当中后，打开服务商开关。
    

![](https://docs.cherry-ai.com/~gitbook/image?url=https%3A%2F%2F3562065924-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F0Ut5BptC3t8CtSU1UWpM%252Fuploads%252Fgit-blob-0e0f24a7a8dcf233ad76e40699d9098a97f16597%252Fimage%2520%288%29%2520%282%29.png%3Falt%3Dmedia&width=768&dpr=4&quality=100&sign=ec9888ad&sv=2)

### 

[hashtag](#kai-tong-bing-tian-jia-mo-xing)

开通并添加模型

*   在方舟控制台侧栏最下方的 [开通管理arrow-up-right](https://console.volcengine.com/ark/region:ark+cn-beijing/openManagement?LLM=%7B%7D&OpenTokenDrawer=false) 开通需要使用的模型，这里可以按需开通豆包系列和 DeepSeek 等模型。
    

![](https://docs.cherry-ai.com/~gitbook/image?url=https%3A%2F%2F3562065924-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F0Ut5BptC3t8CtSU1UWpM%252Fuploads%252Fgit-blob-df1f031bf318217d967d5559c5c9e9a5b62ff349%252Fimage%2520%281%29%2520%281%29%2520%282%29%2520%281%29.png%3Falt%3Dmedia&width=768&dpr=4&quality=100&sign=73738808&sv=2)

*   在 [模型列表文档arrow-up-right](https://www.volcengine.com/docs/82379/1330310#%E6%96%87%E6%9C%AC%E7%94%9F%E6%88%90) 里，找到所需模型对应的 模型ID。
    

![火山引擎模型ID列表示例](https://docs.cherry-ai.com/~gitbook/image?url=https%3A%2F%2F3562065924-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F0Ut5BptC3t8CtSU1UWpM%252Fuploads%252Fgit-blob-a93ede69c85506e59181c40d40aea56f94c764f5%252F%25E7%2581%25AB%25E5%25B1%25B1%25E5%25BC%2595%25E6%2593%258E_%25E6%25A8%25A1%25E5%259E%258BID.png%3Falt%3Dmedia&width=768&dpr=4&quality=100&sign=e0b937cf&sv=2)

*   打开 Cherry Studio 的 [模型服务](/pre-basic/settings/providers) 设置找到火山引擎
    
*   点击添加，将之前获得的 模型ID 复制至 模型ID 文本对话框即可
    

![](https://docs.cherry-ai.com/~gitbook/image?url=https%3A%2F%2F3562065924-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F0Ut5BptC3t8CtSU1UWpM%252Fuploads%252Fgit-blob-66ed3f0b8eb33b5aa64b9969325930283b0b8c0e%252Fvolc_ark_01.png%3Falt%3Dmedia&width=768&dpr=4&quality=100&sign=57805e7c&sv=2)

*   按照此流程依次添加模型
    

### 

[hashtag](#api-di-zhi)

API地址

API地址有两种写法

*   第一种为客户端默认的：`https://ark.cn-beijing.volces.com/api/v3/`
    
*   第二种写法为：`https://ark.cn-beijing.volces.com/api/v3/chat/completions#`
    

circle-info

两种写法没什么区别，保持默认即可，无需修改。

关于 `/` 和 `#` 结尾的区别参考文档服务商设置的 API 地址部分，[点击前往](/pre-basic/settings/providers#api-di-zhi)
