# 新手指南

欢迎来到 Contract 平台！只需3步，即可开始审计智能合约。

## 🚀 快速开始

### 第1步：连接钱包

1. 点击页面右上角的 **「连接钱包」** 按钮
2. 选择您常用的钱包（推荐 MetaMask）
3. 在钱包弹窗中确认连接请求
4. 签名登录消息（免费，无需 Gas）

> 💡 **提示**：首次登录时系统会自动为您创建账户

### 第2步：输入合约地址

1. 在首页搜索框中粘贴合约地址（以 0x 开头）
2. 选择合约所在的区块链网络
3. 点击 **「开始审计」**

### 第3步：查看审计报告

审计完成后，您将看到完整的安全报告，包括：

- **安全评分**：0-100 分，分数越高越安全
- **漏洞列表**：按严重程度排序的问题清单
- **修复建议**：针对每个漏洞的解决方案
