square-code贡献代码
===============

我们欢迎对 Cherry Studio 的贡献！您可以通过以下方式贡献：

1\. **贡献代码：**开发新功能或优化现有代码。

2\. **修复错误：**提交您发现的错误修复。

3\. **维护问题：**帮助管理 GitHub 问题。

4\. **产品设计：**参与设计讨论。

5\. **撰写文档：**改进用户手册和指南。

6\. **社区参与：**加入讨论并帮助用户。

7\. **推广使用：**宣传 Cherry Studio。
